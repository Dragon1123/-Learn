Css

  1. 元素边框模拟三角形

    border: 块级元素的边框默认是由四个梯形构成，当宽度和高度没有时，将由边框填充满整个元素，这时边框就变成了四个等腰三角形
    eg:

    div{ width:0; hight:0; border:1px sol red; }

Javascript

  1. 获取元素宽度，高度和边距

    offsetwidth/offsetheight: 获取元素宽度和高度（包括padding、border）
    eg:

    var e=docment.getElementById('e');
    e.offsetwidth;
    e.offsetheight;

    offsettop/offsetleft: 获取的边距（元素与父元素之间的距离）
    eg:

    var e=docment.getElementById('e');
    e.offsettop;
    e.offsetleft;

  2. 地址描述
